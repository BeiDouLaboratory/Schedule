# Schedule
自己新建分支，内容直接写在 `README` 中，一个星期（建议）按照以下格式 `commit` 一次，大家互相监督学习(。・∀・)ノ

> 分支：[branch] -------------- 名字不做要求，能辨认即可

### 格式

> **[H2]** 年份 月份

> 内容：
> 
>- 学习计划
>学习C语言
>看python
>备战英语四级




### Tips

> 所有内容在填写之后即可删除（除 Schedule）

> 每个月份的记录不必删除，直接在顶部继续按照学习情况追加到同一个 `README` 文件即可，不必再新建文件
