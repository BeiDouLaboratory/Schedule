# Schedule
自己新建分支，内容直接写在 `README` 中，一个星期（建议）按照以下格式 `commit` 一次，大家互相监督学习(。・∀・)ノ

> 分支：[branch] -------------- 名字不做要求，能辨认即可

### 格式

> **[H2]** 年份 月份

> 内容：
> 
>- 计划实践
>  - 这个星期是否有阅读专业相关书籍？
>     - 如阅读过，列举书籍、阅读量
>     - 知识吸收得如何
>     - 是否有相关的笔记，或者脑图
>   - 这个星期是否有相关实践？
>     - 如有，简要列举
>   - 对下个星期的安排（可选）
> - 疑问（可选）
>   - [问题归属]
>      - 具体内容



- Tips
  - 以上模板内容在填写之后即可删除（除 Schedule）
  - 每个月份的记录不必删除，直接在顶部继续按照学习情况添加到同一个 `README` 文件即可，不必再新建文件
