# Schedule
自己新建分支，内容直接写在 `README` 中，一个星期（建议）按照以下格式 `commit` 一次，大家互相监督学习(。・∀・)ノ

### 格式

> 分支：[branch] -------------- 名字不做强制要求，能辨认即可

> 内容：
> [yyyy-MM-dd] Commit
>
>- 计划实践
>   - 这个星期是否有阅读专业相关书籍？
>     - 如阅读过，列举书籍、阅读量
>     - 知识吸收得如何
>     - 是否有相关的笔记，或者脑图，可填写 `Blog` 地址
>   - 这个星期是否有相关实践？
>     - 如有，简要列举
>   - 对下个星期的安排（可选）
>- 疑问（可选）
>    - [问题归属]
>      - 具体内容
>- 计划实践
>  - 学习了简单的数组知识
>   - 二分法查找，链表，栈，一些排序算法，在leetcode上练习算法，学习了一些函数
>- 了解了一些GitHub的一些使用
>- 熟悉和使用了VMware
